# ReadDistance

Read distance between the sensor and other object in meters per second.

## Library reference

* [SwiftIO](https://github.com/madmachineio/SwiftIO)
* [MadBoard](https://github.com/madmachineio/MadBoards)
* [HCSR04](https://github.com/madmachineio/MadDrivers/tree/main/Sources/HCSR04/HCSR04.swift)


## How to use

1. Download the MadDrivers folder.
2. Open and run one of the examples in Visual Studio Code. If you are not familiar with the operation, you could refer to [this tutorial](https://docs.madmachine.io/overview/advanced/run-example).
